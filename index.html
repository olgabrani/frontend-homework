Javascript CSS/Animations Stuff
Simple carousel
Assume you have a list of 5 images, each one with dimensions 300x200px.
Create a simple carousel, with 2 control buttons that move the main image back and forth.
When the first image is visible, the "back" button should be disabled. When the last image is visible, the "front" button should be disabled.

For extra points:

  • Modify your code to work with X images
  • Use infinite back and forth buttons
Modal
Create a link that when clicked a modal is launched.
To close this modal you have to click on a "close" icon.

For extra points:

  • Close the modal on clicked outside its area
  • Use different animation methods to open the modal
  • Create two modals
JQuery plugin
Create a simple JQuery plugin, that once included in a webpage, will alert an annoying "hello world" on document ready.

For extra points:

  • Allow the plugin user to change the "hello world" with a custom one
  • Create a plugin for the carousel with options for the animating effect
